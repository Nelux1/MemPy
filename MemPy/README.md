# MemPy

# Ejecución y Requerimientos
Para ejecutar la aplicación primero hay que instalar los requerimientos.

## Requerimientos

### Unix/macOS
```python -m pip install -r requirements.txt```

### Windows
```py -m pip install -r requirements.txt```

## Ejecución

### Unix/macOS
```python run.py```

### Windows
```py run.py```
